This Application (DesksideToolkit) is primarily used by Deskside Support Technicians within a specific environment.
The application uses qtDesigner and Python, written in PyQt5. 

QTDesigner is used to create the buttons and additional slots & Signals. Install QTDesigner and open the DesksideToolkit.ui This will show you the layout used, signals & slots and how they're connected as well.
Using VSCode (or any IDE), open the DesksideToolkit.py and voila, you then have the Python side of the program (Written using PyQt5).
